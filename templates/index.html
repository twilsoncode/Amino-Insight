Amino Insight Creation


The Amino Insight website is built using a combination of a flask python app, html, CSS and JavaScript. The python flask app acts as a framework, communicating between the inputs for each tool and the python function to return a value. The CSS used in this website is based on Mobirise styling, you can learn more about their styles here.
